🌾 Farm2Table
📌 Overview
Farm2Table is a platform that helps restaurants buy directly from farmers, eliminating middlemen.
It ensures farmers earn better income while restaurants receive fresh and quality products.

🚀 How It Works
Restaurants place orders directly from farmers
Each village has a representative
The representative:
Manages communication between farmers and restaurants
Collects products from farmers
Handles delivery to restaurants
🎤 Representative Selection
Candidates apply for the village representative role
They go through an interview process
The best candidate is selected based on performance
💡 Key Features
✅ Direct farmer-to-restaurant connection
✅ No middlemen
✅ Better income for farmers
✅ Fresh products for restaurants
✅ Job opportunities in villages
✅ ai Interview-based representative selection
🎯 Objectives
Improve farmers' earnings
Provide fresh food supply
Create employment in rural areas
